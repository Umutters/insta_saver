## Özellikler

- Instagram post, reels ve story indirme
- Linkten otomatik medya algılama
- Cihaz galerisine kaydetme
- Modern ve kolay arayüz
- Hızlı indirme ve ilerleme göstergesi

---

## Gereksinimler

- Flutter SDK (öneri: güncel stable)
- Android Studio / Xcode (hedef platforma göre)
- Bir Android/iOS cihaz veya emulator/simulator

---

## Kurulum ve Çalıştırma

```bash
git clone https://github.com/Umutters/insta_saver.git
cd insta_saver
flutter pub get
flutter run
```

## Kullanım

1. Instagram'da indirmek istediğiniz post/reel/story bağlantısını kopyalayın.
2. Uygulamada ilgili alana yapıştırın.
3. **İndir** butonuna basın.
4. Medya cihazınıza kaydedilir.
